## ğŸ§­ Methodology (Summary)

1. **Compilation**  
   Aggregated hubs from public sources (official websites, directories, ecosystem listings).

2. **Verification & Cleaning**  
   - Website and/or public contact validated  
   - Duplicates removed  
   - Names, cities, and countries normalized  

3. **Classification**  
   Rule-based hub type classification:
   - Innovation / Tech Hub  
   - Incubator  
   - Accelerator  
   - Coworking Space  
   - Makerspace / FabLab  
   - University / Research Hub  
   - NGO / Foundation  

4. **Geocoding**  
   Latitude and longitude generated using reputable geocoding services.
